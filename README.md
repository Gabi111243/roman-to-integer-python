Roman numerals are composed of the following seven symbols:

| Symbol | Value |
|--------|-------|
| I      | 1     |
| V      | 5     |
| X      | 10    |
| L      | 50    |
| C      | 100   |
| D      | 500   |
| M      | 1000  |

### Special Cases:

- I before V or X (e.g. IV = 4, IX = 9)
- X before L or C (e.g. XL = 40, XC = 90)
- C before D or M (e.g. CD = 400, CM = 900)
